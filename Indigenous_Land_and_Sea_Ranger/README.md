# Indigenous Land and Sea Ranger

## Project Overview
The Indigenous Land and Sea Ranger project aims to support Indigenous communities in managing and protecting their land and sea resources. This application provides tools for monitoring, reporting, and engaging with the community regarding environmental stewardship.

## Project Structure
The project is divided into two main parts: the backend and the frontend.

### Backend
The backend is built using Node.js and Express. It handles API requests, manages data, and connects to a database.

- **Entry Point**: `backend/src/app.js`
- **Controllers**: `backend/src/controllers/index.js`
- **Routes**: `backend/src/routes/index.js`
- **Models**: `backend/src/models/index.js`
- **Package Configuration**: `backend/package.json`
- **Documentation**: `backend/README.md`

### Frontend
The frontend is developed using React. It provides a user-friendly interface for interacting with the backend services.

- **Main Component**: `frontend/src/App.jsx`
- **Reusable Components**: `frontend/src/components/index.jsx`
- **Home Page**: `frontend/src/pages/Home.jsx`
- **Entry Point**: `frontend/src/index.js`
- **Package Configuration**: `frontend/package.json`
- **Documentation**: `frontend/README.md`

## Setup Instructions
1. **Clone the Repository**
   ```
   git clone <repository-url>
   cd Indigenous_Land_and_Sea_Ranger
   ```

2. **Backend Setup**
   - Navigate to the backend directory:
     ```
     cd backend
     ```
   - Install dependencies:
     ```
     npm install
     ```
   - Start the backend server:
     ```
     npm start
     ```

3. **Frontend Setup**
   - Open a new terminal and navigate to the frontend directory:
     ```
     cd frontend
     ```
   - Install dependencies:
     ```
     npm install
     ```
   - Start the frontend application:
     ```
     npm start
     ```
